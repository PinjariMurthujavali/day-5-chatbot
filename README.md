\## 📌 About This Project



Part of my \*\*AI 90-Day Challenge\*\* — building one real AI project every single day.



Day 5 takes the chatbot beyond simple conversation and adds \*\*self-analysis capabilities\*\* — the bot now understands its own conversations.



\## ✨ Features



\- 📊 \*\*Chat Statistics\*\* — total messages, word count, character count

\- 😊 \*\*Sentiment Analysis\*\* — detects positive / negative / neutral tone in real-time

\- 🔑 \*\*Keyword Extraction\*\* — identifies top discussed topics automatically

\- 💾 \*\*JSON Export\*\* — saves structured analysis report after every session

\- 🎭 \*\*Multiple AI Personalities\*\* — mentor, comedian, strict teacher, zen master, enthusiast



\## 🛠️ Tech Stack



\- \*\*Language:\*\* Python 3.x

\- \*\*AI/LLM:\*\* Groq API (Llama 3.3 70B)

\- \*\*Libraries:\*\* `python-dotenv`, `json`, `datetime`



\## 🚀 How to Run



```bash

\# Clone the repo

git clone https://github.com/PinjariMurthujavali/day-5-chatbot.git

cd day-5-chatbot



\# Setup virtual environment

python -m venv venv

venv\\Scripts\\activate



\# Install dependencies

pip install groq python-dotenv



\# Add your Groq API key in a .env file

GROQ\_API\_KEY=your\_key\_here



\# Run the chatbot

python chatbot.py mentor

```



Available personalities: `mentor`, `comedian`, `strict\_teacher`, `zen\_master`, `enthusiast`
